HTML and CSS Web Resources

1: W3Schools:

W3Schools is a great resoruce when it comes to learning about HTML and CSS, I've linked to the base page for HTML but they have tons of resoruces that assist you in learning HTML and CSS. I like this website for its great examples.

2: TutorialsPoint:

TutorialsPoint is a good resoruce to learn about HTML5, it has many tutorials, I like it for its tutorials on HTML syntax.

3: Learn HTML:

Learn HTML like the rest on the list is a great resoruce when it comes to leanring HTML. This website is especially good at showing you HTML code then showing exactly what that code will do.

4: Wev.dev:

Wev.Dev is a very in depth resoruce for learning HTML, it goes over a lot of the basics and goes over it in more depth then most other resources would.

5: Scaler:

Lastly, Scaler, scaler is a great resoruce, it has text and videos so you can choose which you want based on your preferences. It also has a good tutorial on how to use VS code for HTML
